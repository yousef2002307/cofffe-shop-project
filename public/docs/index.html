Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

1 Security

APIs for user registration and login and logout

Register a new user

Example request:
curl --request POST \
    "http://localhost/api/v1/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"totam\",
    \"email\": \"schinner.angelita@example.net\",
    \"password\": \"LkW-ibQ7QTR\",
    \"phone\": \"voluptas\"
}"

const url = new URL(
    "http://localhost/api/v1/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "totam",
    "email": "schinner.angelita@example.net",
    "password": "LkW-ibQ7QTR",
    "phone": "voluptas"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "message": "User registered successfully"
}

You can check the Dev Tools console for debugging information.

Get a paginated list of products

No auth required. If the quantity of a product is 0 or less, you cannot buy or add it to the cart. Pagination: add ?page parameter to the request.

    --header "Content-Type: application/json" \
